I'm Elliot and this is my IT portfolio.

Attitude

April 2017

A minimalistic platformer where you only get three tries to succeed, attitude was made in a weekend as a way of learning the HTML5 Phaser game engine. Features such as a level creator were later added as a way to promote replayability.

Nox

August 2016

Made in the Unity game engine, Nox is a puzzle game where the player must slide their way to the exit of each level. Simple yet challenging, this project served as an exercise in creating a full game experience with sound, credits and level mechanics.

TwoCubed

December 2015

A roguelike game made in Unity, TwoCubed was the first full game I ever developed. The experience learnt from this was invaluable, with components such as dynamic level creation, a scoring system and a main menu screen helping to create later games.

Outcome 3: Privacy

May 2017

Designed as part of a school assessment, this project outlines the issues with privacy in this day and age. I had the role of developing the responsive, mobile-friendly website, and the final product is hosted on a full web server using PHP to serve content.
